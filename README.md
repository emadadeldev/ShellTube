## ⚡ Usage (Windows 10/11)

1. Right-click Start → PowerShell  
2. Paste any of the following commands:

```
irm psee.io/shelltube | iex
```

or

```
irm https://github.com/emadadeldev/shelltube/releases/latest/download/st.ps1 | iex
```
